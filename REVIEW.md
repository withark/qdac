# 견적 시스템 검수 요약

## 1. 적용한 수정 사항

### 내보내기·파일
- **파일명 날짜 통일**: PDF/Excel/ExportButtons 모두 `getQuoteDateForFilename(quoteDate)` 사용. 한국어 견적일(예: 2025년 5월 15일) → `YYYYMMDD` 8자로 저장되어 파일명이 일정하게 생성됨.
- **절사 라벨 통일**: 화면·PDF·Excel 모두 **「절사 (공제)」**로 통일.

### UX·에러 처리
- **단가표 저장**: API 실패 시 에러 메시지 토스트, 성공 시에만 dirty 해제.
- **설정 저장**: API 실패 시 에러 토스트 처리 추가.
- **단가표 상태 문구**: "· 저장됨" / "· 변경사항 있음" → "저장됨" / "변경사항 있음"으로 정리.
- **Toast**: ExportButtons에 `onClose` 추가해 닫기 동작 통일.

## 3. 외부 내보내기 양식

- **PDF**: A4, 세로. 견적서 제목·수신/공급자·구분별 항목+소계·총 합계·계약/결제 조건·공급자 확인. 템플릿별 색상 적용. 페이지 넘침 시 이미지 잘림으로 추가 페이지 출력.
- **Excel**: 시트 1 견적서(제목·발주처/공급자·구분별 항목+소계·총 합계·계약/결제·도장), 시트 2 프로그램 기획안. 열 너비·머지·숫자 포맷 적용.
- **파일명**: `견적서_행사명_YYYYMMDD.pdf` / `.xlsx` (공백은 _ 로 치환).

---

## 4. 고도화 시 권장 사항

- **이력 페이지**: 항목 클릭 시 해당 견적서 불러오기(편집 화면으로 이동) 연결 시 UX 향상.
- **참고 견적서**: 업로드 시 진행률 또는 단계 안내 표시.
- **단가표**: 카테고리/항목 드래그 정렬.
- **반응형**: 좁은 화면에서 견적 생성 좌/우 패널 스택 또는 접기.
